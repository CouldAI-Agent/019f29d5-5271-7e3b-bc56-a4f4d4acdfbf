# Gestão de Estoque e Paletes

Um aplicativo Flutter para gerenciamento de estoque em armazéns. O sistema permite cadastrar racks (filas de estantes), prateleiras e gerenciar os paletes alocados em cada espaço, facilitando a localização rápida de produtos.

## Funcionalidades
- **Gestão de Racks**: Adicione novas filas de racks ao armazém.
- **Gestão de Prateleiras**: Adicione prateleiras a cada rack.
- **Gestão de Paletes**: Registre paletes com identificações (ex: "Limão 06-124") informando o produto e o lote/código.
- **Busca Rápida**: Pesquise por produto ou lote e descubra exatamente em qual rack e prateleira o palete está localizado.
- **Visão Geral**: Visualize a estrutura do seu armazém, navegando entre racks, prateleiras e os paletes alocados.

## Tecnologias
- Flutter
- Dart

## Como Executar
1. Certifique-se de ter o Flutter instalado.
2. Clone o repositório.
3. Execute `flutter pub get` para instalar as dependências.
4. Execute `flutter run` para iniciar o aplicativo no seu dispositivo ou emulador preferido.
